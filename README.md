# Basic Webpage Design using HTML5

## 📌 Objective
To design a basic webpage using HTML5 elements such as:
- Text Formatting
- Lists (Ordered, Unordered, Description)
- Hyperlinks (Internal, External, Email)
- Marquee
- Images

## 🛠️ Tools Used
- HTML5
- Web Browser

## 📂 Files Included
- index.html

## 📖 Description
This project demonstrates the use of basic HTML5 elements.
It includes formatted text, different types of lists, hyperlinks,
an image, and a marquee element.

## 🌐 Output
The webpage displays structured content using HTML5 tags.
